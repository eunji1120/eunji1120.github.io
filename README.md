## Featured Project

### Bayesian Heavy-Tailed Modeling of Wildfire PM2.5

**Repository:** [UBC STAT 405 Project](https://github.com/eunji1120/UBC_Stat_405_project)

A solo Bayesian modeling project analyzing wildfire-driven extreme air pollution risk in British Columbia’s Southern Interior.

The project models approximately four years of daily PM2.5 data using Stan, comparing a Normal seasonal baseline against a Student-t model to better capture heavy-tailed wildfire smoke events.

**Key methods used:**

- Bayesian modeling in Stan
- Hamiltonian Monte Carlo inference
- Student-t heavy-tailed modeling
- LOO-CV model comparison
- Posterior predictive checks
- Prior sensitivity analysis
- Synthetic-data calibration study
- HMC vs ADVI comparison

**Why it matters:**  
Air pollution risk is not only about average PM2.5 levels. Wildfire smoke creates extreme tail events, and those events require careful modeling, validation, and uncertainty-aware interpretation.
